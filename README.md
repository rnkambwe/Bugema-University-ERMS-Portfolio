## Project Overview

The **Bugema University Educational Resource Management System (ERMS)** is a web-based enterprise information system used by **Bugema University** to support core academic and administrative operations. The platform enables prospective students, current students, faculty, and administrative staff to manage academic processes through a centralized online portal.

The ERMS supports functions such as:

- Student admissions
- Student registration
- Course registration
- Academic records management
- Student information management
- Administrative reporting

The project was undertaken to improve operational efficiency, reduce reliance on paper-based processes, and provide secure, centralized access to institutional records.
